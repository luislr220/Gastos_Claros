# GastosClaros: Sistema de Control Financiero Inteligente

**GastosClaros** es una aplicación de gestión de finanzas personales diseñada para eliminar la incertidumbre sobre el destino del presupuesto mensual. El sistema se enfoca en la integridad de los datos y la automatización mediante una arquitectura limpia.

## Problema y Solución
Muchos usuarios pierden el rastro de sus "gastos hormiga" debido a procesos de registro tediosos o datos desordenados. **GastosClaros** soluciona esto mediante:
* **Registro Ágil:** Entrada simplificada de nombre y monto.
* **Categorización Inteligente:** Algoritmo que identifica palabras clave para asignar categorías automáticamente (ej. "Starbucks" ⮕ "Comida").

## Stack Tecnológico
* **Backend:** Node.js con Express.
* **FrontEnd:** Angular
* **Arquitectura:** Patrón de capas (Routes, Controllers, Services, Models).
